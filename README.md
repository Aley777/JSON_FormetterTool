# JSON Formatter Tool

A modern developer tool for formatting, validating, minifying, and visualizing JSON data.

Built with React and designed with a clean, professional UI.

---

## Features

- Format JSON (pretty print)
- Minify JSON
- Validate JSON syntax
- JSON Tree View (expand / collapse)
- Dark / Light mode
- Copy formatted output
- Download JSON file
- Drag and Drop JSON upload
- Auto-format on paste
- Fetch JSON from URL
- Character and line count
- Disabled actions for better UX
- Toast notifications

---

## Tech Stack

- React (Vite)
- CSS
- @uiw/react-json-view

---

## Installation

    npm install
    npm run dev

---

## Usage

Format JSON  
Paste JSON and click Format to make it readable.

Minify JSON  
Use the Minify button to remove whitespace.

Validate JSON  
Check if your JSON is valid.

Tree View  
Switch to Tree view for collapsible structure.

Drag and Drop  
Drop a .json file directly into the input panel.

Auto Paste  
Paste JSON and it formats automatically.

Fetch from URL  
Enter a JSON API URL and click Fetch.
