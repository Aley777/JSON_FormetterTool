# JSON Formatter Tool

A clean and simple developer tool for formatting, validating, minifying, and copying JSON data.

## Features

- Format messy JSON
- Minify JSON
- Validate JSON syntax
- Copy formatted output
- Load sample JSON
- Choose indentation size
- Character counter
- Responsive layout

## Tech Stack

- React
- Vite
- CSS

## Getting Started

```bash
npm install
npm run dev
